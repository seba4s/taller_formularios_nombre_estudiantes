# Taller Formularios Django

Proyecto del Taller 2 en Django con trabajo por ramas.

## Flujo de ramas acordado

1. Rama base: main
2. Rama de integracion: develop
3. Ramas de trabajo desde develop:
   - feature/asistencia
   - feature/solicitudes

## Flujo para asistencia

1. Trabajar y hacer commits en feature/asistencia.
2. Abrir Pull Request de feature/asistencia hacia develop.
3. Validar funcionalidad y aprobar cambios.
4. Hacer merge de develop hacia main cuando todo el taller este completo.

## Aplicacion implementada

La aplicacion asistencia incluye:

- Modelo Asistencia
- Formulario AsistenciaForm (ModelForm)
- Vista de registro
- Vista de confirmacion
- URLs de la app e integracion al proyecto
- Registro en panel de administracion
